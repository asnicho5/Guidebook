How to Use the Prompt Browser:
The prompt browser is a way for language learners to browse examples of prompts they can use to boost their learning with ChatGPT.

To use the browser, simply filter by your target language, your language level, and the prompt type you would like to view, and the prompts will be filtered by these parameters. You can also search for a specific topic, language, level, or prompt type by using the search bar.

You can click on a prompt card and view more detailed information about the prompt, such as the prompt text and output. You can then copy the prompt and try it out in ChatGPT yourself, or modify certain parts about the prompt to fit your specific needs.
Prompt Cards:
Each prompt has its own "card" that users can click on to view more detailed information about the prompt.

On each card, users can view information such as the prompt title, the prompt language, the prompt language level, and the prompt type.

Green cards indicate "Any" language level,
Blue cards indicate the "Beginner" language level,
Yellow cards indicate the "Intermediate" language level,
and Red cards indicate the "Advanced" language level.

On each prompt card, you will also notice an icon. The icons visually indicate the prompt type (such as "Choose Your Own Adventure" or "Content Generation").
