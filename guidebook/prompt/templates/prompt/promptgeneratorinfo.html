How to Use the Prompt Generator:
The prompt generator is a way for language learners to make their own prompts to copy and paste into ChatGPT.

Steps for using the generator:

1. Select your target language
2. Select a prompt category
3. Input your specific parameters for the prompt
4. Generate the prompt
5. Copy and paste the prompt into ChatGPT

Congratulations, you've just generated your own prompt!
