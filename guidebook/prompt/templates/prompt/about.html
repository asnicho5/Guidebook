About

The purpose of this website is to help language learners harness the power of ChatGPT to aid in their language learning. It contains 3 sections: the Guidebook, the Prompt Browser, and the Prompt Generator.

In the guidebook, students can review the various sections of information about how to use ChatGPT for language-learning purposes. Some sections include the "Accuracy Disclaimer" section, which goes over some of the issues encountered through experimentation with ChatGPT's language capabilities, the "Prompt Types" section, which describes the a few types of prompts that students can use, and the "Indicating Level in a Prompt" section, which shows examples of prompts at different language levels.

In the prompt browser, students can browse various existing prompts and a sample ChatGPT output.

In the prompt generator, students can input their own parameters and generate their own prompts to copy and paste into ChatGPT.
